# Countdown to 2026

A pixel-art New Year's countdown with canvas, fireworks and a night sky background.

## Getting Started
You have 2 methods to view the site:

1. Download the files and open index.html 

2. Go to https://loek.oerlemans.tv/countdown_2026

## Files
- `index.html` – page structure (canvas, labels, happy-new-year overlay)
- `style.css` – night sky background, stars, moon, label styling
- `countdown.js` – logic for countdown, pixel digits, rockets and fireworks
- `images/fav.png` – an image of pixel art 2026

## Interaction
- Click on the canvas or background for extra fireworks.

## Features
- Dynamic removal of days/hours/minutes when they reach 0, remaining digits scale up.
- Slow color shift only on colons; other digits remain stable.
- Rockets with trailing lines that explode into fireworks.
- Sporadic yellow shooting stars.
- Large "Happy New Year" overlay with extra fireworks at midnight.
- Countdown digits explode into colorful fireworks when transitioning to zero.
- Beautiful gradient night sky with twinkling stars and glowing moon.
- Smooth animations and particle effects throughout the countdown.
- Customizable graphics quality settings for optimal performance on different devices.
